# 🌎 Air Quality Chatbot

Streamlit 기반 대기질 정보 조회 및 LLM 챗봇.

```bash
# 가상 환경 설정
python -m venv venv
source venv/bin/activate  # (Mac/Linux)
venv\Scripts\activate     # (Windows)

# 필요 라이브러리 설치
pip install -r requirements.txt

# Ollama 서버 실행 (CMD 또는 PowerShell에서 별도 실행)
ollama serve

# Streamlit 실행
streamlit run air-chatbot.py

# 가상 환경 종료
deactivate
```
